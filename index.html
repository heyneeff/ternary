The Balance Proof
N coins, one heavier. A balance scale returns 3 outcomes:
N left pan heavier   Z balanced   P right pan heavier

Each weighing is a native BRANCH3. Binary needs 2 comparisons for the same 3-outcome decision.
Optimal: ⌈log₃(N)⌉ weighings — provably unreachable by binary.
