The Balance Proof
N coins, one is heavier. A balance scale returns 3 outcomes:
N  left pan heavier    Z  balanced    P  right pan heavier

Each weighing is a native BRANCH3 operation.
Optimal result: ⌈log₃(N)⌉ weighings — provably unreachable by binary.
